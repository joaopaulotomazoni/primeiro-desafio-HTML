  1. Entendendo comunicação Client X Server

    • História da Web

      Neste módulo, começamos estudando a primeira geração de computadores. O Harvard Mark I foi criado em 1944 e foi um dos primeiros computadores eletromecânicos de grande escala, desenvolvido nos Estados Unidos durante a Segunda Guerra Mundial.

    • O que são Clients

      Cliente-Servidor por Tiago de Jesus Neves está licenciado sob CC BY-SA 3.0.

      Durante a aula, aprendemos que clients são os dispositivos que solicitam e recebem recursos de um servidor, interagindo diretamente com o usuário.

    • O que são Servers

      Servers representam o lado do servidor, ou seja, o sistema que fornece recursos — como dados (em servidores de banco de dados), serviços ou programas — para outros dispositivos. O servidor pode ser desde um computador comum em nossa casa até algo mais robusto, como os datacenters de grandes empresas, que atendem em larga escala.

    • Linguagens de programação

      Aprendemos que existem linguagens de programação utilizadas tanto no lado server quanto no lado client. Alguns exemplos de linguagens utilizadas exclusivamente no lado client são HTML e CSS. No lado do servidor, podemos citar linguagens como Python, Java, C, C++ etc. Há também linguagens que podem ser usadas em ambos os lados, como o JavaScript.

  2. Introdução ao HTML na prática

    • Inspetor de elementos

      O inspetor de elementos é uma ferramenta dos navegadores web que permite os usuarios vizualizarem, manipularem os códigos-fonte das HTML, CSS e JavaScript das páginas web. Normalmente é possível acessar está ferramente utilizando a tecla F12.

    • Estrutura básica do HTML

      Nesta aula aprendemos sobre as tags basicas como:

      "html:" é a raiz do documento HTML;

      "head": é o cabeçalho, onde fica as informações que não ficam no conteúdo principal da página;

      "title": é o titulo da pagina no navegador;

      "body": é o corpo do texto, onde fica o conteúdo a ser exibido na página.

    • Falando Sobre Tags

      Nesta aula aprendemos sobre as tags abaixo:

      "i": serve para colocar o texto em itálico;

      "strong": serve para deixar o texto em negrito;

      "input": serve para adicionar uma caixa de texto, senha, data, etc;

      "img" serve para inserir imagens.

    • Falando sobre atributos

      Atributos em HTML sãoo palavras especiais utilizadas dentro de Tags de abertura para modificar ou fornecer informações adicionais sobre um elemento. Como por exemplo a Tag "img" e seu atributo "src=" que serve para atribuir o endereço da imagem.

    • Textos

      Abaixo vou será exibido as Tags de texto aprendidas nesta aula:

      "h": as tags "h" servem para titulos ou subtitulos e vão do tamanho 1, sendo a maior letra, até 6, sendo o menor tamanho;

      "p": servem para separar os paragrafos de um texto;

      "blockquote": serve para colocar uma observação ou citação;

      "sup": serve para colocar um item na parte superior de um texto, como por exemplo um número ao quadrado (2 2);

      "mark": serve para destacar algo no texto;

    • Listas ordenadas e não ordenadas

      As listas ordenadas servem para numerar tópicos ou separa-los com um indicador:

      "ol": esta tag serve para criar listas ordenadas, com numeros indicando a ordem;

      "ul": esta tag serve para criar listas sem ordem definida, apenas com marcadores;

      "li" esta tag serve para iniciar um texto dentro das listas, iniciando um novo item.
